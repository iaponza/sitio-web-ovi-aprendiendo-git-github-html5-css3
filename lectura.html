Objetivo general: Para la creación de este OVI (Objeto Virtual de Información) es importante tener en cuenta que este recurso digital es diseñado para brindar una mejor información a los usuarios, a fin de que puedan manipular y administrar sistemas de control de versiones como GIT o GITHUB, Exhibiendo y dando a conocer el paso a paso necesario para diseñar un sitio web y los repositorios, facilitando así el acceso y comprensión de la información por parte de la persona interesada.
Objetivo específico 1: Facilitar el acceso y compresión del sistema de control de versiones y editores de códigos web, al momento de crear repositorios y/o sitios web.
Objetivo específico 2: Ofrecer ayuda a los usuarios para que puedan entender y utilizar los lenguajes de Diseño web como HTML5 y CSS3, sus estructuras y aplicación de elementos de HTML5 y CSS3 a proyectos de diseño Web.
Objetivo específico 3: A través de este Objetivo Virtual de Información se podrán visualizar documentos, videos, tutoriales y una serie de información que será de gran ayuda a la gran cantidad de usuarios que consulten nuestro repositorio, Organizar lógicamente el procedimiento necesario para diseñar un sitio web con fuentes verídicas y siempre protegiendo los derechos de autor.
